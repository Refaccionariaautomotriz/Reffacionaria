Clutch

Beneficios

  • Resistencia térmica y mecánica
  • Proporciona cambios suaves para un fácil manejo
  • Excelente fricción para una transferencia de potencia más eficiente
  • Especificaciones de equipo original
  • Kit completo (plato, disco y collarín)
  • Disco elaborado con resinas sin asbesto
  • Balanceo propuesto

Tips de mantenimiento

Normalmente el clutch se deberá cambiar únicamente cuando este se daña, algunas señales de falla suelen ser cuando:

  • Patina
  • Vibra
  • No corta
  • Hace ruido
